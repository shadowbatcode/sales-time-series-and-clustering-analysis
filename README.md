# Sales Time Series And Clustering Analysis

来源：
- `国赛/23国赛C`

项目内容：
- `data_preprocess.py`
  主预处理和分析脚本
- `question1_visualization.py`
  综合可视化脚本
- `graph.py`
  销售网络图构建脚本
- 同目录保留了原始附件、预处理结果、图表、图网络文件和中间结果

项目特点：
- 这个项目的数据和结果文件较完整，适合直接作为一次完整分析记录保存
- 代码与数据文件大多位于同一层级，保留原相对路径后可读性最好

体积说明：
- 仓库中保留了较大的 `xlsx` 与 `gexf` 文件
- 目前单文件仍低于 GitHub 100 MB 限制，但如果你想更轻量，我可以再帮你拆成 `raw-data` 与 `results` 两版
